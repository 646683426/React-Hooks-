## React Hooks的个人理解
Hook 就是 JavaScript 函数，但是使用它们会有两个额外的规则：
1、只能在函数最外层调用 Hook。不能在循环、条件判断或者子函数中调用。
2、只能在 React 的函数组件中或自定义的Hook调用 Hook。不能在其他 JavaScript 函数中调用。
